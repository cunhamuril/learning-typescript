# Aprendendo TypeScript

## Comandos básicos

```bash
$ yarn add -D typescript # Instalar em forma de desenvolvimento, pois em produção o TS vai ser convertido pra JS

$ yarn add -D @types/express # Instalar definição de tipagens

$ yarn tsc <arquivo> # Converter código para JS. **Antes de gerar arquivo de configuração
# Exemplo:
$ yarn tsc src/index.ts

$ yarn tsc --init # Gerar arquivo de configuração do TS

$ yarn tsc # Converter todos os códigos TS para JS
```
